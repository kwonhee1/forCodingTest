# 최댓값과 최솟값

[코딩테스트 연습 - 최댓값과 최솟값 | 프로그래머스 스쿨](https://school.programmers.co.kr/learn/courses/30/lessons/12939)

## 문제 요구 사항 분석  

1. 주어진 문자열의 숫자들 중 최솟값 최댓값을 구하시오

## 사용 알고리즘  

 1. 없음

## 입력 사항

1. 문자열 s

## 출력 사항

1. (최솟값 최댓값)

## 입출력 예시

입력 1

> "1 2 3 4"

출력 1

> "1 4"

입력 2

> "-1 -2 -3 -4"

출력 2

> "-4 -1"

## 풀이 방법

1. 문자열 s를 " "으로 구분하여 나눈다
2. 나눈 문자열을 숫자로 변경한다
3. max min을 최신화 한다
